simulation d'un Proxy en  java c'est un  serveur qui situé entre deux partie ( un client et 3 ervant ) 

servant 1 : qui  propose une connection  avec UDP
servant 2 : qui  propose une connection  avec middleware RMI 
servant 2 : qui  propose une connection  avec un web service implémenté en JAVA

pour executer  l'application il fauts :
1- importer la Base de Données vente  par limportation du  fichier "vente.sql"
2- importer le projet  dans  eclipse 

3- executer  le serveur de SI_Ent1 
4- executer  le serveur de SI_Ent2
5- executer  le serveur de SI_Ent3
6- executer  le serveur de Proxy
7- executer  le Client .
